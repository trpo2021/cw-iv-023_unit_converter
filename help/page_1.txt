Входные данные необходимо ввести по шаблону:
"Category(<unit in> <unit from>, <value>)".
Перечень ключей и вариации запуска программы с ними:
  <-f/--fast>  - ключ для быстрого запуска программы без запрооса на ввод.
По окончании работы программы будет выведена переведенная величина.
Для запуска программы с этим ключем используется другой шаблон ввода:
Category <unit in> <unit from>, value.

Пример запуска программы:
bin/unit_converter --fast Length km m, 10
