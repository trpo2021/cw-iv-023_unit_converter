Входные данные необходимо ввести по шаблону:
"Category(<unit in> <unit from>, <value>)".
Перечень ключей и вариации запуска программы с ними:

  <-f/--fast>  - ключ для быстрого запуска программы без запроса на ввод.
По окончании работы программы будет выведена переведенная величина.
Для запуска программы с этим ключем используется другой шаблон ввода:
Category <unit in> <unit from>, value.

Пример запуска программы:
bin/unit_converter --fast Length km m, 10

 <-g> - ключ для подключения графического интерфейса. При запуске программы с
данным ключем откроется окно, в котором из выпадающих списков вы можете выбрать
категорию и величины, относящиеся к данной категории. В поле ввода вы можете 
ввести значение величины, которую нужно перевести. Полученное после перевода 
значение выведется ниже поля ввода.
